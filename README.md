## Run Locally

Clone the project

```bash
  git clone https://github.com/Akshay0831/Institution-Accreditation-and-Automation-System.git
```

Keep two terminals,

To install dependencies,

In one terminal,

```bash
  cd backend
  npm install
```

In another terminal,

```bash
  cd frontend
  npm install
```

Then, in both terminals,

```bash
  npm start
```

To add the Database,

From the root folder of the project

```bash
  mongorestore ./projectdb/dump/
```
